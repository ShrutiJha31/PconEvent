# LineUp

> Event Scheduler

FullCalendar , GoogleCalendar API, react , redux , mongodb , express

### Install server dependencies

```bash
npm install
```

### Install client dependencies

```bash
cd client
npm install
```

### Run both Express & React from root

```bash
npm run dev
```
